# AdiRentCar

Aplikasi Rental Mobil berbasis Java Desktop menggunakan Apache NetBeans dan MySQL.

## Fitur

- Login Admin
- Dashboard
- Manajemen Data Mobil
- Manajemen Data Pelanggan
- Transaksi Penyewaan
- Pengembalian Mobil
- Perhitungan Denda
- Laporan Data

## Tools

- Java
- Apache NetBeans
- MySQL (XAMPP)
- Swing GUI

## Cara Menjalankan

1. Import database `adirentcar`
2. Buka project menggunakan Apache NetBeans
3. Jalankan aplikasi
