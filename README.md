# 🚗 AdiRentCar

Aplikasi Rental Mobil berbasis **Java Desktop (Swing)** yang dikembangkan menggunakan **Apache NetBeans** dan **MySQL (MariaDB)**.

Project ini dibuat sebagai tugas perkuliahan sekaligus media pembelajaran implementasi CRUD, transaksi database, serta pengembangan aplikasi desktop menggunakan Java.

---

## 📖 Deskripsi

AdiRentCar merupakan aplikasi yang membantu proses pengelolaan usaha rental mobil, mulai dari pengelolaan data kendaraan, data pelanggan, transaksi penyewaan, hingga proses pengembalian kendaraan.

---

## ✨ Fitur

- ✅ Dashboard
- ✅ Manajemen Data Mobil
- ✅ Manajemen Data Pelanggan
- ✅ Transaksi Penyewaan
- ✅ Pengembalian Mobil
- ✅ Status Mobil (Tersedia / Disewa)
- ✅ CRUD Data
- ✅ Koneksi Database MySQL
